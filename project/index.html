Diagrams.net Setup

  1. Visit app.diagrams.net
  2. Create new diagram → Select Flowchart template
  3. Use the workflow structure shown in the demo
  4. Add decision diamonds for profile completion checks
  5. Include parallel messaging processes (WhatsApp, SMS, Email)
  6. Create the 3-day reminder loop with fallback logic
  7. Export and share as public link

Key Workflow Elements

  • Start: User Signup Trigger
  • Parallel: Multi-channel Welcome Messages
  • Decision: Profile Completion Status
  • Loop: 3-day Reminder Cycle
  • Fallback: WhatsApp → SMS Logic
  • End: Multiple Exit Points
